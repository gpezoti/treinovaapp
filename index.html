Configurar Supabase

Para o app funcionar com login, feed e gerenciamento de alunos, é preciso conectar a um projeto Supabase (grátis).

  1. Crie um projeto em supabase.com
  2. No SQL Editor, rode o arquivo supabase-setup.sql (entregue junto)
  3. Em Project Settings → API, copie a Project URL e a anon public key
  4. Cole nas variáveis SUPABASE_URL e SUPABASE_ANON no topo do <script> deste arquivo
  5. Recarregue a página

Veja o README.md para o passo a passo completo.
